Вставьте URL опубликованной таблицы

⚠️ Таблица не опубликована

Чтобы приложение могло читать данные, таблицу нужно опубликовать:

  1. Откройте вашу Google Таблицу
  2. Перейдите в Файл → Поделиться → Опубликовать в интернете
  3. Выберите нужный лист
  4. Нажмите Опубликовать
  5. Вернитесь сюда и нажмите "Подключить" снова

Важно: "Доступ по ссылке" — это НЕ то же самое. Нужно именно Опубликовать в интернете через меню Файл.

Как опубликовать Google Sheet

  1. Откройте ваш Google Sheet
  2. Перейдите в Файл → Поделиться → Опубликовать в интернете
  3. Выберите нужный лист для публикации
  4. Нажмите Опубликовать
  5. Вернитесь сюда и нажмите "Повторить"

Важно: "Доступ по ссылке" — это не то же самое, что "Опубликовать в интернете". Нужно именно опубликовать через меню Файл.
